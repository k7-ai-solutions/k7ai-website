# K7 AI website

GitHub Pages-ready static website.

1. Upload all files inside this folder to your repository root.
2. Keep `index.html` in that root.
3. GitHub: **Settings → Pages → Deploy from a branch → main → /(root)**.

Before launch, replace `hello@k7ai.in` and the `wa.me` URLs in `index.html` with your own email and WhatsApp number.
